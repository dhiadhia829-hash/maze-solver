+Maze Solver

A Python project that generates a random maze and solves it using different search algorithms.

+Features
- Random maze generation
- BFS (Breadth-First Search)
- DFS (Depth-First Search)
- A* algorithm
- Simple text output of the maze and path

+What I learned
- How pathfinding works on grids
- Basic graph traversal
- Using queues, stacks, and heaps

+How to run
python maze_solver.py
